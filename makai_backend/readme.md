# Makai Backend Setup

## Virtual Environment Setup

To create the virtual environment:
```bash
py -m venv venv
```

To activate the virtual environment:

**Git Bash:**
```bash
source venv/Scripts/activate
```

**PowerShell:**
```powershell
.\venv\Scripts\Activate.ps1
```

**Command Prompt:**
```cmd
venv\Scripts\activate
```
